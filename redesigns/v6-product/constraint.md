# Constraints — v6-product

Hard rules for all copy and code in this redesign. Check every proposed line against this list before presenting it.

## Copy
1. **No em dashes (—).** Never use the em dash character in any copy, headline, subhead, or body. Use a period, comma, colon, or parentheses to restructure the sentence instead. This applies to en dashes used as em dashes too. (Hyphens in compound words like "forward-deployed" are fine.)
