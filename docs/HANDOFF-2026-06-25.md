## Gotchas burned (don't repeat)
- Never assign refs inside `useMemo` (StrictMode double-invoke ‚Üí orphan ‚Üí silent no-op; mutate the rendered object in `useFrame`).
- Bloom needs an opaque scene bg; dim via CSS container opacity only.
- Stay on `three@^0.184`; downgrading silently breaks WebGL render.
- WebGL context exhaustion from many reloads ‚Üí `null 'alpha'` = dev artifact (restart browser), not a bug.
