# FDE / Embed-Build-Operate Companies — Synthesis for Ajolote v5 Redesign

Reference library studying how 13 forward-deployed / embedded-AI / deploy-and-operate companies present themselves on their marketing sites. Goal: inform Ajolote Labs' v5 IA, positioning language, proof patterns, and visual/motion direction. Ajolote constraints to honor throughout: **dark theme, anonymized cases, no founder profile, operator-not-consultancy**.

Each company has its own file in this folder. Quotes marked `[unverified]` could not be confirmed via direct fetch (JS-rendered homepages / 403s) and were sourced from search snippets.

## (b) The 5–7 strongest patterns worth adopting

1. **Lead with the operator verb, not a feature.** Cognition's "Cognition **operates** Devin" and Palantir's FDE "owns everything through post-deployment fixes" are the cleanest analogs to Ajolote's thesis. Ajolote's hero should center an operate/run verb ("We build and **run** the brain") — not "we build AI agents," which reads as delivery.

2. **Anonymized-but-quantified proof wall (Distyl model).** Distyl proves you can carry a credibility section with zero named logos using a metrics grid ("50+ deployments / 1B+ decisions / 80% reduction"). Pair this with the get10xai/11x device of **one metric per card**. This is the single best fit for Ajolote's no-named-cases rule: keep the numbers, drop the names.

3. **Name a productized first stage (Cresta "Automation Discovery").** Cresta turns "discovery" into a concrete, ROI-projecting named step. Ajolote should name and surface its Discovery → Node → Agents → Outward MCP → Maintain stages so a non-technical buyer can see how an engagement works and feel de-risked.

4. **The three-layer engagement triad (Ada "Platform + Practice + Experts").** A clean way to package "data brain + agents + we operate it." Maps almost 1:1 onto Ajolote's "Internal Node / Agents / Outward MCP" — and the "Experts" layer is exactly the FDE/operator story that distinguishes Ajolote from a tool.

5. **"Works with the stack you already have" (Maven AGI).** Directly on-thesis for Ajolote's build-on-existing-stack, no-rip-and-replace positioning. Pair with Cresta's "shared memory never lost across handoffs" to describe the unified Node/brain. Reassures ops buyers who fear a costly migration.

6. **Continuity-as-the-product language (Mechanical Orchard).** "We run the system while we incrementally modernize it" sells maintenance-as-the-business better than anyone in the set. Ajolote should make ongoing operation the explicit, repeated stake — the headline benefit, not a footnote.

7. **The get10xai single-page IA spine + dark/cyan visual treatment.** Hero → stat chips → logo+metric strip → 3 capability cards → case carousel → numbered 3-step process → stat-chip CTA is a clean, scannable flow that maps onto Ajolote's story, and the dark near-black + glowing-cyan + serif-headline-with-accent-word look is already on-brand. Add Cresta's **video-background hero** as the cinematic motion cue and 11x's interactive demo moment (hear/see a real agent).

## (c) Anti-patterns to avoid

1. **Named-logo brag-walls and attributed exec testimonials** (Sierra, Decagon, Tribe, Maven, Cresta, Tomoro). Violates Ajolote's anonymized-cases stake — and Ajolote can't field Mattel/Fidelity-tier names anyway. Use anonymized metric cards instead.

2. **Delivery-speed chips as the lead** ("48hr deploy / 21 days to launch" — get10xai; "production in weeks" — Tribe; "in production in <12 weeks" — Tomoro). Speed framing actively undercuts long-horizon operator positioning, and collides with Ajolote's hard stop: **never estimate time to the client.** Keep speed-as-credibility *feeling* without a literal number; replace launch chips with continuity/maintenance metrics.

3. **Light / generic enterprise-SaaS palettes** (Decagon, Ada, Tribe, Maven, likely Cresta). They blend into the category. Ajolote's dark, infrastructure-grade look is the differentiator — lean in.

4. **Burying the buyer CTA under recruiting** (Cognition's "Explore roles" is the most prominent button; Tomoro's "Join Us"). Ajolote must keep its single Calendly CTA primary — hiring lives elsewhere.

5. **Over-grandiose / hype copy** ("Rearchitecting industries" — Distyl; "From zero to 10x" — get10xai). Too abstract or too cheap for a non-technical owner/COO. Ajolote should stay concrete and operational ("we run the system that does X for your business").

6. **Founder/team-pedigree credibility** (Distyl's ex-Palantir/OpenAI lean; Tomoro's "in alliance with OpenAI"). Violates Ajolote's no-founder-profile stake. The moat is operating the client's data brain, not a lab badge.

## (d) Closest analogues to Ajolote's operator model

1. **Distyl AI — the true twin.** "AI-Native Operations," forward-deploys engineers who "own the outcome," lifecycle language ("strategy → deployment → monitoring → continuous improvement"), dark/institutional, and — uniquely — an **anonymized metrics-only proof wall with no named logos.** This is the single most directly transferable reference for tone, theme, and proof. Borrow heavily; just drop the "rearchitecting industries" grandiosity and the team-pedigree lean.

2. **Mechanical Orchard — the operator-continuity reference.** The only company in the set that makes "we run it while we keep improving it" the explicit pitch ("operate, and innovate continuously"). Its principles → process → named-platform (Imogen®) spine is the best template for selling maintenance-as-the-business. Ajolote's "Node" can play Imogen's productized-credibility role. (Avoid its proof vacuum and light palette.)

3. **Cresta — the closest on engagement-framing + motion.** "Automation Discovery" is a productized discovery-then-deploy-then-operate arc, "unified platform for human + AI" reassures ops buyers, "integrate into any environment / shared memory across handoffs" maps to the unified Node, and the **video-background hero** is the one genuinely cinematic reference matching Ajolote's dark-motion instinct. (Strip the named logos and metric-overload.)

**Honorable mention — Palantir & Cognition** as the *tonal* north stars: the operator-not-vendor gravitas and the "we operate [X]" headline verb. And **Anthropic's Applied AI FDE language** (MCP servers / sub-agents / agent skills "in production," "white-glove deployment support," relationships "throughout the lifecycle of an engagement") is the strongest *copy* steal for an MCP-native operator — see `openai-anthropic-fde.md`.
