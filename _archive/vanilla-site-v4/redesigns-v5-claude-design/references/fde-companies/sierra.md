# Sierra
URL(s) read: https://sierra.ai

- **Positioning:** Agent platform ("Sierra Agent OS") for customer experience. Category claim: the place where great companies' AI agents are built and run.
- **Hero:** Headline "Better customer experiences. Built on Sierra." Subhead "Sierra helps the great companies of the world show up at their best."
- **IA (section order):** Hero → leading-brand logo wall → "Transform your customer experience" (3 value props) → customer testimonials/results → Sierra Agent OS product → Ghostwriter (agent that builds agents) → Insights/optimization → Agent Data Platform → trust/certifications → final CTA → footer.
- **Proof pattern:** Massive named-logo wall (Rocket Mortgage, Vanguard, SoFi, Nubank, ADT, Sonos, Ramp, etc.) + named-exec testimonials. E.g. Caryn Seidman-Becker (CLEAR CEO): "Innovation is in our DNA, and Sierra helps us bring that to life." Brand-name density is the whole credibility play.
- **Engagement framing:** Strongly ongoing-operation, not one-time. Continuous improvement via Ghostwriter updates, Insights monitoring, Experiments, proactive workflows. The agent is a managed, optimized asset — close to Ajolote's "operate it" thesis.
- **Visual/motion:** Dark theme, clean/minimalist, enterprise-trustworthy. Blue/purple accents, dashboard-screenshot imagery, certification badges.
- **CTA:** "Learn more" (soft, repeated). No aggressive sell — confidence-by-restraint.
- **Steal / Avoid for Ajolote:** STEAL — dark+trust aesthetic, ongoing-optimization narrative (mirrors operator positioning), named-exec testimonial format. AVOID — the giant logo wall (Ajolote anonymizes cases) and the soft "Learn more" CTA; Ajolote's single calendly CTA is sharper for a smaller, consultative pipeline.
